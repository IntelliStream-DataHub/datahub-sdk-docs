# Healthcare — pharmacy cold chain

**The problem.** Vaccines, blood products and many medications are only safe within a
narrow temperature band. A pharmacy or ward fridge that drifts out of range — a door
left ajar, a failing compressor overnight — can spoil tens of thousands of euros of
stock and, worse, leave a patient given a product that quietly lost its potency. Staff
can't watch every fridge; they need an alarm the moment one strays, and a clean
temperature record to prove the chain held.

What we solve here is protecting temperature-sensitive stock with live monitoring and an
auditable record.

## 1. Watch every fridge live

Each fridge streams its temperature; a subscription brings them into one feed. When one
leaves its band, raise a `cold_chain_excursion` event so it's attended before stock is
lost. See [Consume live data](/guides/realtime-subscriptions).

<Tabs groupId="lang">
<TabItem value="java" label="Java">

```java
import ai.intellistream.datahub.sdk.subscriptions.SubscriptionListener;
import ai.intellistream.datahub.sdk.subscriptions.SubscriptionMessage;
import java.time.Duration;

try (SubscriptionListener listener =
         client.subscriptions().listen(List.of("pharmacy_fridges"))) {
    while (running) {
        SubscriptionMessage msg = listener.poll(Duration.ofSeconds(10));
        if (msg == null) continue;
        if (outOfBand(msg.payload(), 2.0, 8.0)) {     // °C — vaccine fridge band
            EventModel excursion = new EventModel();
            excursion.setExternalId("cold_chain_excursion_f12_" + System.currentTimeMillis());
            excursion.setType("cold_chain_excursion");
            excursion.setStatus("critical");
            excursion.setEventTime(ZonedDateTime.now());
            excursion.setMetadata(Map.of("fridge", "fridge_pharmacy_12", "temp_c", "9.3"));
            client.events().create(List.of(excursion));
        }
        listener.ack(msg.messageId());
    }
}
```

```rust
use dataplatform_rust_sdk::events::Event;

let mut listener = api.subscriptions.listen(&["pharmacy_fridges"]).await?;
while let Some(Ok(msg)) = listener.next().await {
    if out_of_band(&msg.payload, 2.0, 8.0) {           // °C — vaccine fridge band
        let mut excursion = Event::new("cold_chain_excursion_f12".into());
        excursion.r#type = Some("cold_chain_excursion".into());
        excursion.status = Some("critical".into());
        api.events.create(&vec![excursion]).await?;
    }
    listener.ack(&[msg.message_id.as_str()]).await?;
}
```
