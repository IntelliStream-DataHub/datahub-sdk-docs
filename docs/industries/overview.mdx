# Industry scenarios

Thirty worked examples, each starting from a real problem an industry needs solved and
showing how to solve it with the SDK. They're grouped below by the **kind** of problem,
so you can jump to the closest match to yours.

:::tip Two depths
The scenarios on this page are **quick starts (~5 min)** — read one, copy the shape,
adapt the ids. When you're ready to go past CRUD into feature engineering and
prediction, see **[Advanced scenarios](/advanced/predictive-maintenance)** (~20 min to
2 hours each).
:::

## Live monitoring & alerting

React to data as it lands — a push connection, a rule, an alert in time to act.

- [IT operations](/industries/it-operations) — alert on an SLO breach the instant it happens
- [Healthcare](/industries/healthcare) — a live capacity wall-board before a ward overflows
- [Aquaculture (salmon)](/industries/aquaculture) — catch an oxygen crash in time to aerate
- [Maritime](/industries/maritime) — a reefer cold-chain alarm mid-ocean
- [Smart buildings](/industries/smart-buildings) — flag a stuffy zone before the complaints
- [Data centers](/industries/data-centers) — catch a thermal problem before racks throttle
- [EV charging](/industries/ev-charging) — know a charger is dead the moment it fails
- [Airports](/industries/airports) — catch a turnaround slipping before the slot is lost
- [Logistics](/industries/logistics) — spot a delivery route falling behind
- [Banking (AML)](/industries/banking) — react to a flagged payment in real time

## Going deeper

The [Advanced scenarios](/advanced/predictive-maintenance) take a few of these further
into real algorithms:

- [Predictive maintenance](/advanced/predictive-maintenance) — learn a machine's healthy
  vibration signature and flag the drift *(~1–2 hr)*
- [Demand forecasting](/advanced/demand-forecasting) — forecast load or sales and store it
  as a series *(~1 hr)*
- [Early warning](/advanced/oxygen-crash-early-warning) — predict a salmon-pen oxygen crash
  before it happens *(~1 hr)*
- [Asset health scoring](/advanced/asset-health-score) — one health index from many
  signals *(~20–30 min)*
- [Fraud classification](/advanced/fraud-classification) — graph features + a model to
  rank fraud/AML alerts by risk *(~1–2 hr)*
