# Industry scenarios

Worked examples, each starting from a real problem an industry needs solved and showing
how to solve it with the SDK. They're organised by **sector** — pick yours and work
through its scenarios, which build on each other across the value chain.

:::tip How long each takes
Every scenario is self-contained: a **Set up demo data** block (run once) → the
solution → **See the result**. Plan on **~5 minutes to read** one, or **~10–15 minutes
to run it end to end** from an empty workspace (a couple of minutes more the very first
time, to install the SDK and set `BASE_URL`/`TOKEN`). When you're ready to go past CRUD
into prediction, the **[Advanced scenarios (AI)](/advanced/machine-learning-gently)** — no
data-science background needed — run ~20 minutes to 2 hours each.
:::

## Going deeper

The [Advanced scenarios (AI)](/advanced/machine-learning-gently) take these further into real
algorithms — forecasting, classification, clustering and anomaly detection. **No
data-science background needed:** start with
[Machine learning, gently](/advanced/machine-learning-gently) for the ideas in plain
language, then [Generate sample data](/advanced/generate-sample-data) for a sandbox to
run them against. To clean and feature-engineer data while keeping every step traceable,
see [Data pipeline & lineage](/advanced/data-cleaning-lineage). The models below are
grouped by what they *do*.

### Detect & monitor condition

Learn what healthy looks like, then surface the drift.

- [Predictive maintenance](/advanced/predictive-maintenance) — flag a machine drifting from its healthy vibration signature *(~1–2 hr)*
- [Asset health scoring](/advanced/asset-health-score) — one composite health index from many signals *(~20–30 min)*
- [LSTM anomaly detection](/advanced/lstm-anomaly-detection) — catch a multivariate event from the joint pattern *(~2 hr)*
- [Process monitoring with PCA](/advanced/pca-process-monitoring) — spot a break in many sensors' normal relationship *(~1–1.5 hr)*

### Forecast & predict ahead

Predict what's coming while there's still time to act.

- [Demand forecasting](/advanced/demand-forecasting) — forecast load or sales with gradient boosting *(~1 hr)*
- [LSTM forecasting](/advanced/lstm-forecasting) — forecast a decline curve with a neural network *(~2 hr)*
- [Early warning](/advanced/oxygen-crash-early-warning) — predict a salmon-pen oxygen crash before it happens *(~1 hr)*
- [Failure prediction with XGBoost](/advanced/xgboost-failure-prediction) — predict an ESP failure a week out, with feature importance *(~1–2 hr)*

### Classify, cluster & infer

Label, group or estimate from features.

- [Fraud classification](/advanced/fraud-classification) — graph features + a model to rank fraud/AML alerts *(~1–2 hr)*
- [K-Means clustering](/advanced/kmeans-clustering) — discover operating regimes, asset cohorts and network communities *(~1–1.5 hr)*
- [Soft sensor with Random Forest](/advanced/random-forest-soft-sensor) — infer a lab quality from cheap sensors *(~1 hr)*
