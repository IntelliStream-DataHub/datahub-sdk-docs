# Healthcare — hospital operations & patient flow

**The problem.** A hospital runs at the edge of capacity. Beds free up and fill
minute by minute, the emergency department backs up, and a ward that quietly hits
100% occupancy turns into a corridor of waiting patients. The operations team needs a
single live picture of capacity across every ward — and an alert the moment a unit is
about to overflow, while there's still time to act.

This scenario is about **live operational monitoring**: model the hospital, stream
occupancy in, and drive a capacity wall-board off a push connection. It uses only
operational signals — bed counts, wait times, census — never patient records.

## 1. Model the hospital

A hospital *contains* wards, a ward *contains* beds. The same hierarchy you'd model
for any physical estate, which lets a ward-level alert roll up to the whole site.

## 2. Drive the capacity wall-board live

A wall-board polling every minute is already stale. Subscribe to the occupancy series
and update the moment a reading lands — and raise a `capacity_warning` event when a
ward crosses its threshold, so charge nurses are paged before it's full. See
[Consume live data](/guides/realtime-subscriptions) for the listener lifecycle.

<Tabs groupId="lang">
<TabItem value="java" label="Java">

```java
import ai.intellistream.datahub.sdk.subscriptions.SubscriptionListener;
import ai.intellistream.datahub.sdk.subscriptions.SubscriptionMessage;
import java.time.Duration;

try (SubscriptionListener listener =
         client.subscriptions().listen(List.of("bed_capacity"))) {
    while (running) {
        SubscriptionMessage msg = listener.poll(Duration.ofSeconds(2));
        if (msg == null) continue;
        updateWallboard(msg.payload());
        if (overThreshold(msg.payload())) {            // e.g. occupancy > 95%
            EventModel warning = new EventModel();
            warning.setExternalId("capacity_warning_icu_" + System.currentTimeMillis());
            warning.setType("capacity_warning");
            warning.setStatus("open");
            warning.setEventTime(ZonedDateTime.now());
            warning.setMetadata(Map.of("ward", "ward_icu", "occupancy_pct", "96"));
            client.events().create(List.of(warning));
        }
        listener.ack(msg.messageId());
    }
}
```

use dataplatform_rust_sdk::events::Event;

let mut listener = api.subscriptions.listen(&["bed_capacity"]).await?;
while let Some(Ok(msg)) = listener.next().await {
    update_wallboard(&msg.payload);
    if over_threshold(&msg.payload) {                  // e.g. occupancy > 95%
        let mut warning = Event::new("capacity_warning_icu".into());
        warning.r#type = Some("capacity_warning".into());

    }
    listener.ack(&[msg.message_id.as_str()]).await?;
}
