# DataHub SDK documentation

The external-facing documentation site for the DataHub SDK (Java, Python, Rust),
built with [Docusaurus](https://docusaurus.io/). Code samples use synced language
tabs — pick a language once and every snippet on the site follows.

## Develop

Requires Node.js >= 20.

```bash
npm install        # first time only
npm start          # dev server with hot reload (http://localhost:3000)
```

To preview on this host's LAN address (e.g. for review on another device):

```bash
npm start -- --host 0.0.0.0 --port 8001
```

## Build

```bash
npm run build      # static site into ./build
npm run serve      # serve the production build locally
```

## Layout

- `docs/` — Markdown / MDX content. With `routeBasePath: '/'`, `docs/intro.md`
  (slug `/`) is the landing page; the sidebar is autogenerated from the folder tree.
- `src/css/custom.css` — the IntelliStream-branded GitBook-style skin (Infima vars
  for light/dark, brand teal `#105984`, title treatment, code-block and tab styling).
- `static/img/` — `logo.svg` / `logo-dark.svg` (light/dark navbar logo) and `favicon.ico`.
- `docusaurus.config.js` — site config: navbar, footer, offline search
  (`@easyops-cn/docusaurus-search-local`) and Prism languages (java, rust, python, …).

Versioned docs are added later via `npm run docusaurus docs:version <x.y>`.
